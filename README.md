# androGolang

Andro yang ditulis dengan Golang supaya lebih laju

## Install

Clone repositori ini.

Pastikan sudah punya Golang.

Build dengan perintah `go build`.

## Init Project

```bash
andro init
```

## Build Project

```bash
andro
```

## Catatan

Untuk absolute URL, gunakan base `/assets/`.
